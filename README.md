# Single Page CV

Este é um projeto de currículo web simplificado (Single Page Application), desenvolvido para apresentar informações profissionais de forma clara e responsiva em uma única página.

## 🚀 Tecnologias Utilizadas

*   **HTML5**: Estruturação semântica do conteúdo.
*   **CSS3**: Estilização e layout responsivo.

## 📋 Seções do Projeto

O arquivo principal `index.html` está organizado nas seguintes seções:

*   **Cabeçalho**: Informações de contato e links sociais.
*   **Sobre**: Breve resumo profissional.
*   **Experiência**: Histórico profissional detalhado (âncora `#experiencia`).
*   **Formação**: Educação acadêmica e cursos.
*   **Habilidades**: Competências técnicas e ferramentas.

## 🛠️ Como Visualizar

1. Clone este repositório:
   ```bash
   https://github.com/jailson1987/Single-pagevc/commit/91fff76bb87669be2d6a6e83b9d098bc6d2f1b3d
   ```
